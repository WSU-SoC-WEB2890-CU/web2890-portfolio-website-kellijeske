Project 2

Wavy Portfolio

This portfolio website showcases my design skills and professional experience through a clean and modern layout. The site features custom vector graphics that align with the color theme, adding visual harmony throughout. Smooth, subtle animations enhance user engagement without overwhelming the content, while wavy transitions between sections offer a creative yet professional touch. Built using Bootstrap and SCSS, the site is fully responsive, ensuring an optimal viewing experience across all devices. The design project reflects a proficiency in front-end development, providing visitors with useful information about me and my work.

I particularly enjoyed implementing background textures in this project-- for example, the simple but interesting artwork that is incorporated into the review section.
